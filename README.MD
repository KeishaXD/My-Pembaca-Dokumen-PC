# My Pembaca Dokumen

My Pembaca Dokumen adalah aplikasi desktop (PC) berbasis **Electron + JavaScript** untuk membaca novel dan dokumen digital secara offline.

## Fitur (Tahap Awal)
- Aplikasi desktop berbasis Electron
- Berjalan di Windows (Linux & macOS direncanakan)
- Format yang direncanakan:
  - PDF
  - EPUB
  - ZIP / CBZ (gambar / manga)
  - PBR / format custom

## Teknologi
- Electron
- Node.js
- HTML, CSS, JavaScript

## Struktur Proyek

My Pembaca Dokumen/
├── node_modules/
├── package.json
├── package-lock.json
├── main.js
├── preload.js
└── index.html

## Cara Menjalankan (Development)

1. Masuk ke folder project  
   cd My Pembaca Dokumen  

2. Install dependency  
   npm install  

3. Jalankan aplikasi  
   npm start  

## Catatan
- Proyek ini masih dalam tahap pengembangan awal
- Fokus utama adalah pembaca dokumen offline
- Tampilan (UI) masih sederhana

## Rencana Pengembangan
- Open file dialog
- Pembaca PDF
- Pembaca EPUB
- Pembaca ZIP / manga
- Library & bookmark
- Mode gelap / terang
